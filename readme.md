# 富文本编辑器功能对比

## 需求功能
* 基本编辑功能：字体设置，字号设置，靠左，靠右，粗体，斜体，划线，列表
* 表格
* 表格增删行列
* 图片上传
* 自定义按钮，部分插件细节不符时可重新定义
